## 🔧 技术实现

### 颜色配置
每个材质包含 4 种颜色，通过渐变和混合创造流体效果：
```swift
case .aurora:
    return [
        Color(red: 0.0, green: 1.0, blue: 0.5), // Green
        Color(red: 0.0, green: 0.8, blue: 1.0), // Cyan
        Color(red: 0.5, green: 0.0, blue: 1.0), // Purple
        Color(red: 1.0, green: 0.4, blue: 0.8)  // Pink
    ]
```

### 混合模式
- 大部分材质使用 `.plusLighter` 混合模式
- Ink 和 Void 使用 `.normal` 模式以保持深色效果

### 模糊效果
- Ice: 20（较锐利）
- Ink: 25（中等）
- Void: 40（柔和）
- 其他: 20（标准）

# 新材质替换说明

## 🎨 替换的材质

### 移除的材质
1. ❌ **墨玉流体（Ink）** - 过于模糊，颗粒感强
2. ❌ **虚空之境（Void）** - 过于模糊，颗粒感强

### 新增的材质

#### 1. ✨ **琥珀之光（Amber）**
**替代**：墨玉流体

**颜色方案**：
- 明亮的琥珀橙：`Color(red: 1.0, green: 0.6, blue: 0.0)`
- 深琥珀：`Color(red: 0.9, green: 0.5, blue: 0.1)`
- 浅琥珀金：`Color(red: 1.0, green: 0.75, blue: 0.3)`
- 暖棕色：`Color(red: 0.8, green: 0.4, blue: 0.0)`

**特点**：
- 温暖的橙金色调
- 模糊半径：18（适中的柔和感）
- 清晰可见，无颗粒感
- 给人温暖、舒适的感觉
- 类似琥珀宝石的光泽

**适用场景**：
- 需要温暖感的时刻
- 秋季氛围
- 放松冥想
- 温馨时光

#### 2. ✨ **星云漩涡（Nebula）**
**替代**：虚空之境

**颜色方案**：
- 明亮的紫色：`Color(red: 0.4, green: 0.2, blue: 0.8)`
- 亮紫罗兰：`Color(red: 0.6, green: 0.3, blue: 1.0)`
- 蓝紫色：`Color(red: 0.3, green: 0.5, blue: 0.9)`
- 粉紫色：`Color(red: 0.8, green: 0.4, blue: 0.9)`

**特点**：
- 绚丽的紫色系
- 模糊半径：22（稍微柔和但清晰）
- 明亮清晰，色彩丰富
- 给人梦幻、神秘的感觉
- 类似宇宙星云的色彩

**适用场景**：
- 需要灵感的时刻
- 夜晚冥想
- 探索内心
- 梦幻体验

## 🎯 设计理念

### 琥珀之光
**灵感来源**：琥珀宝石
- 温暖的橙金色调
- 半透明的质感
- 自然的光泽
- 舒适的视觉体验

**设计目标**：
- 提供温暖、舒适的视觉感受
- 清晰可见，无颗粒感
- 适中的模糊度，保持流体感
- 色彩明亮但不刺眼

### 星云漩涡
**灵感来源**：宇宙星云
- 绚丽的紫色系
- 多层次的色彩变化
- 梦幻的视觉效果
- 神秘而明亮

**设计目标**：
- 提供梦幻、神秘的视觉体验
- 明亮清晰，色彩丰富
- 适度的模糊度，保持星云感
- 紫色系更加明显和吸引人

## 💡 技术优化

### 模糊度平衡
```swift
case .amber: return 18   // 琥珀：适中的柔和感
case .nebula: return 22  // 星云：稍微柔和但清晰
```

**优势**：
- 避免过度模糊（原来的 25 和 40）
- 避免颗粒感（原来的 12）
- 保持流体的柔和质感
- 清晰可见，细节丰富

### 颜色亮度
**琥珀之光**：
- 使用明亮的橙金色
- 避免暗色调
- 高对比度

**星云漩涡**：
- 使用明亮的紫色系
- 多层次色彩
- 丰富的视觉效果

# 墨玉和虚空材质清晰度优化

## 🎯 问题
墨玉（Ink）和虚空（Void）材质过于模糊，看起来像马赛克

## 🔧 解决方案

### 1. 模糊半径优化

#### 修改前
```swift
case .ink: return 25  // 太模糊
case .void: return 40 // 非常模糊
```

#### 修改后
```swift
case .ice: return 15  // 更锐利
case .ink: return 12  // 大幅减少模糊，清晰可见
case .void: return 18 // 从40降到18，保持神秘感但更清晰
default: return 20
```

**改进幅度：**
- 墨玉：25 → 12（减少 52%）
- 虚空：40 → 18（减少 55%）

### 2. 颜色亮度优化

#### 墨玉（Ink）

**修改前：**
```swift
Color(white: 0.3)  // 较暗
Color(white: 0.4)  // 较暗
Color(red: 0.2, green: 0.25, blue: 0.3)
Color(red: 0.15, green: 0.2, blue: 0.25)
```

**修改后：**
```swift
Color(white: 0.4)  // 提高亮度
Color(white: 0.5)  // 更明亮
Color(red: 0.3, green: 0.35, blue: 0.4)  // 浅蓝灰
Color(red: 0.25, green: 0.3, blue: 0.35) // 浅墨蓝色
```

**改进：**
- 整体亮度提升 25-33%
- 保持墨玉的深邃感
- 增强层次感和可见度

#### 虚空（Void）

**修改前：**
```swift
Color(white: 0.05)  // 几乎黑色
Color(red: 0.1, green: 0.0, blue: 0.2)
Color(white: 0.0)   // 纯黑
Color(red: 0.2, green: 0.0, blue: 0.4)
```

**修改后：**
```swift
Color(white: 0.15)  // 提高基础亮度
Color(red: 0.2, green: 0.1, blue: 0.3)  // 更亮的深紫
Color(white: 0.1)   // 不再是纯黑
Color(red: 0.3, green: 0.1, blue: 0.5)  // 更亮的紫色
```

**改进：**
- 整体亮度提升 50-200%
- 保持神秘的深空感
- 紫色更加明显和清晰

## 💡 设计理念

### 平衡清晰度与风格
1. **墨玉**：保持东方墨色的深邃感，但提升可见度
2. **虚空**：保持深空的神秘感，但让紫色更加明显

### 技术考虑
1. **模糊半径**：降低到合理范围，避免过度模糊
2. **颜色亮度**：提升基础亮度，增强对比度
3. **层次感**：通过颜色变化创造深度

# 澄域官网

澄域是一款专注于正念练习的应用，帮助用户在数字时代找到内心的平静与力量。

## 网站内容

- **首页**: 品牌故事、产品理念、功能介绍
- **隐私政策**: 数据收集和使用说明
- **用户协议**: 服务条款和用户权利

## 技术栈

- 纯静态 HTML/CSS
- 响应式设计
- 部署在 Vercel

## 本地预览

直接用浏览器打开 `index.html` 即可。

或使用简单的 HTTP 服务器：

```bash
python3 -m http.server 8000
# 访问 http://localhost:8000
```
